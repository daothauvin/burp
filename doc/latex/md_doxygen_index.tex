Le but de ce projet est d’implémenter en langage C un jeu de programmation dans le cadre du cours de C\+P6. Dans ce jeu, des robots s’entretuent dans une arène, chacun d’eux exécutant un script donné en début de partie. Le jeu se termine lorsqu’il ne reste plus qu’un seul robot encore fonctionnel dans l’arêne \+: son script est alors déclaré gagnant. Le langage de script que nous utiliserons est une version très simplifiée de l’antique langage B\+A\+S\+IC. Le programme devra permettre la simulation de l’arène, ainsi que l’interprétation en temps réel du script de chaque robot.\hypertarget{md_doxygen_index_autotoc_md3}{}\doxysection{Dépendances}\label{md_doxygen_index_autotoc_md3}
